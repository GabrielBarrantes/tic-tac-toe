# Tic tac toe

This is a small web application developed to showcase the ReactJS framework capabilities. Players take turns to play the tic-tac-toe game.

## Setup

```bash
$ git clone git@github.com:GabrielBarrantes/tic-tac-toe.git
$ cd tic-tac-toe/
$ npm install
```

## Run

```bash
$ npm run dev
```
